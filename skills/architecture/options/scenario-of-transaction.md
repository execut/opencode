# Сценарий транзакции (scenario-of-transaction)

Сценарий транзакции — это шаблон проектирования в разработке программного обеспечения,
который организует бизнес-логику приложения в виде набора процедурных сценариев. Каждый такой
сценарий отвечает за обработку одного конкретного запроса, поступающего из слоя представления
(например, из веб-приложения или API).

## Обязательные опции
нет

## Принципы

В данном шаблоне есть только один слой, где лежит код. Структура:
- `<context>/` - папка контекста или реализации приложения
    - `service.go` — реализация контекста или приложения, обращения к БД
    - `...go` - другие вспомогательные файлы
