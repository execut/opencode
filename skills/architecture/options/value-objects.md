# Объекты-значения (value-objects)

Обязательные опции: нет

Обеспечивает согласованность данных в рамках специальных объектов-значений. Теперь в домен будут попадать только проверенные значения,
поскольку конструкторы объектов-значений пробрасывают ошибки при попытке создать объект в невалидном состоянии.
- Теперь все аргументы конструкторов и методов сущностей должны оборачиваться в объекты-значения.
- Вся валидация из других слоёв должна переехать на объекты-значения.
- Все значения бизнес-событий тоже должны быть объектами-значениями

## В комбинации с другими опциями
### optimistic-locking
Версия заворачивается в VO Version, создать который невозможно при отрицательной версии.
